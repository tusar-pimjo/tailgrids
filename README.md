# TailGrids

![hero](apps/docs/src/app/opengraph-image.png)

**TailGrids** is a collection of high-quality, pre-built UI components for [Tailwind CSS](https://tailwindcss.com/). It's designed to help you build beautiful and responsive websites and applications faster, without having to start from scratch.

## Documentation

Visit http://tailgrids.com/docs to view the documentation.
