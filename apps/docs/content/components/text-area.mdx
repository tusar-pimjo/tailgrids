# Text Area

React Text Area components are used to collect longer, multi-line text input from users. They are commonly used for messages, descriptions, feedback forms, comments, and notes where a single-line input is not sufficient.

This Text Area component is built with React and styled using Tailwind CSS. It supports labels, helper text, validation states, and works in both controlled and uncontrolled form setups.

## Usage

Import the component and pass the required props.

```tsx

export default TextAreaExample = () => (
  <TextArea label="Message" placeholder="Write your message..." />
);
```

## API Reference

### TextArea

Extends `textarea` element props.

| Prop        | Type                                | Default     | Description                             |
| ----------- | ----------------------------------- | ----------- | --------------------------------------- |
| `label`     | `string`                            | `-`         | Optional label displayed above textarea |
| `hint`      | `string`                            | `-`         | Hint text displayed below textarea      |
| `state`     | `'default' \| 'error' \| 'success'` | `'default'` | Visual validation state                 |
| `className` | `string`                            | `-`         | Additional classes for customization    |

## Accessibility

- Label is automatically associated using a generated `id` and `htmlFor`.
- Uses semantic `<textarea>` element.
- Fully keyboard accessible.
- Disabled state is communicated visually and programmatically.
- Validation feedback is available through visible messaging.
