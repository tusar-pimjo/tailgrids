# List

Lists display items in a structured, easy-to-scan format.

## Usage

```tsx
import { List } from "@/registry/core/list";

<List>
  <ListItem>Item 1</ListItem>
  <ListItem>Item 2</ListItem>
  <ListItem>Item 3</ListItem>
</List>;
```
