# Radio Input

Radio buttons allow users to select one option from a set.

## Usage

```tsx
import { RadioInput } from '@/registry/core/radio-input';

<RadioInput name="option" value="1" label="Option 1" />
<RadioInput name="option" value="2" label="Option 2" />
```
