# Spinner

Spinners indicate an ongoing process or loading state.

## Usage

```tsx
import { Spinner } from "@/registry/core/spinner";

<Spinner />;
```

## Variants

```tsx
<Spinner variant="default" />
<Spinner variant="dotted" />
<Spinner variant="dotted-round" />
```
