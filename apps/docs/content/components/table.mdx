# Table

Tables display data in a structured format with rows and columns.

## Usage

```tsx
import { Table } from "@/registry/core/table";

<Table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John Doe</td>
      <td>john@example.com</td>
    </tr>
  </tbody>
</Table>;
```
