# Popover

Popovers display additional content in a floating container.

## Usage

```tsx
import { Popover } from "@/registry/core/popover";

<Popover>
  <PopoverTrigger>Open</PopoverTrigger>
  <PopoverContent>Popover content here</PopoverContent>
</Popover>;
```
