# Date Picker

Date pickers allow users to select dates from a calendar interface.

## Usage

```tsx
import { DatePicker } from "@/registry/core/date-picker/single-date";

<DatePicker />;
```

## Range Date Picker

```tsx
import { RangeDatePicker } from "@/registry/core/date-picker/range-date";

<RangeDatePicker />;
```
