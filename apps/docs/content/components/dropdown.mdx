# Dropdown

Dropdowns display a list of choices on temporary surfaces.

## Usage

```tsx
import { Dropdown } from "@/registry/core/dropdown";

<Dropdown>
  <DropdownItem>Option 1</DropdownItem>
  <DropdownItem>Option 2</DropdownItem>
</Dropdown>;
```

## Accessibility

- Keyboard navigation
- Focus management
- ARIA attributes
